Notes
On the right hand side there is a form with Tigra Calendars attached to the fields. Note, that you can have multiple calendars on the same page independently configurable.
Advantages

Easy date input - With Tigra Calendar date input is simple and intuitive. Visitors can select any date from unlimited range with few clicks.

Cost Saving Component - Don't waste your time coding date conversion and validation all over again for each new project. Get the dates on server side ready for further processing.

Professional Look - Simple great looking improvement to the user interface of your web application.
